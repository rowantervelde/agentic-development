## Notes on Data Usage

1. **Attribution**: Most sources request citation. Vektis requires "bron: Vektis". CBS data is free to reuse under CC0/open government license with attribution.
2. **Privacy**: All aggregate/open data has been privacy-protected (minimum group sizes). Individual-level data (CBS Microdata, Nivel research data) requires formal agreements.
3. **Language**: Most portals are Dutch-language first; CBS StatLine has a full English interface. RIVM has partial English content.
4. **API Integration**: CBS OData API and Vektis CSV downloads are the best starting points for programmatic access. R package `cbsodataR` and Python package `cbsodata` simplify CBS API access.
5. **Mergers in progress**: GIPdatabank merging into Zorgcijfersdatabank (announced Feb 2026). Check both URLs during transition.
